# Royal Pro Services Website

Static website for Royal Pro Services Ltd. Built for shared hosting/cPanel with HTML, CSS, JavaScript and PHP contact form support.

## Main Files

- `index.html` - Home page
- `services.html` - Services overview
- `water-damage.html` - Water damage page
- `mold-remediation.html` - Mold remediation page
- `fire-smoke-damage.html` - Fire and smoke page
- `demolition-restoration.html` - Demolition and restoration page
- `contact.html` - Contact page
- `contact.php` - Live form handler for cPanel hosting
- `thank-you.html` - Form success page
